# static getNumberOfLedgerLines(midi, direction, start) → {number}

Calculates the number of ledger lines at the provided midi Assumes that you are asking when you need at least 1
Parameters:
Name Type Description
midi number Midi value of current note
direction String Expected to either be "up" or "down" case insensitive
start String Expected to be either "c" for the bass clef or "a" for the treble clef case insensitive
